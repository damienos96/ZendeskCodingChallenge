#Instructions for adding project to Eclipse and running it
1.	Click FILE -> IMPORT from the Eclipse top menu.
2.	Expand GENERAL, select EXISTING PROJECTS INTO WORKSPACE and click NEXT.
3.	Select the SELECT ARCHIVE FILE option and browse for ZendeskChallenge.zip
4.	Click FINISH.
5.	Before running, open ZendeskChallenge.java and ZendeskChallenge.java and enter a valid Zendesk account domain, email and password
	in quotation marks under the "//Enter your domain, email address and password respectively." comment.
6.	In the package explorer for the project you can run ZendeskChallenge.java for the program,
	or you can run ZendeskChallengeTest.java to run the JUnit tests for the program.
